2. Create an Event (Proposition)

An Event is a contextualized triplet: a statement of fact at a specific time and place.

3. Create a Hop (Hyperedge)

A Hop connects source Events to target Events, forming an inference step.

4. Assemble a Path

A Path is an ordered sequence of Hops, forming a complete argument or causal chain.

ARRAY STRUCTURE | Ontologies stored in ordered array [0...n]. | . INHERITANCE | Index [i] inherits all Concepts/Predicates | | from preceding indices [0...i-1]. | . [ LAYER 2: THE EVENT (PROPOSITION) ] A "Simple Sentence" situating a triplet in space/time. - Triplet: Source --[Predicate]--> Target - Context: Space + Time assertion. - Special: 'is_number' enables numeric input for quantitative data. [ LAYER 3: THE HOP (INFERENCE STEP) ] A "Compound Sentence" acting as a Directed Hyperedge. - Hyper-relational: Connects multiple Sources -> multiple Targets. - Interval Time: Occurs over a duration (e.g., DURING). [ LAYER 4: THE PATH (STATEMENT) ] A "Narrative": An ordered sequence of Hops representing a complete causal chain or formal argument. =============================================================================== [3] STEP-BY-STEP USAGE STEP 1: INITIALIZE ONTOLOGY [ ] Create Ontology (e.g., "Anatomy") [ ] Add Concepts (e.g., "Lungs", "Oxygen") [ ] Add Predicates (Categorize: Triplet, Hop, or Spatial) STEP 2: CREATE EVENTS [ ] Select Source + Target concepts [ ] Define Spatial + Temporal context [ ] Assign Provenance (Textbook, Govt, Deduction) [ ] EXECUTE: [Create Event] -> Stores in active pool. STEP 3: DEFINE HOPS (LOGIC) [ ] Select Source Event(s) (Ctrl+Click for multiple) [ ] Select Target Event(s) [ ] Select Hop Predicate (e.g., "TRIGGERS") [ ] EXECUTE: [Create Hop] STEP 4: ASSEMBLE PATHS [ ] Create New Path [ ] Select Active Path from dropdown [ ] Add Hops in sequence [ ] EXECUTE: [Add Hop to Path] =============================================================================== [4] ANALYSIS AND INFERENCE Utilize the "Second Brain" query functions: +----------------------------+------------------------------------------------+ | FUNCTION | DESCRIPTION | +----------------------------+------------------------------------------------+ | Coalesce Hypernode | Find every Path involving a specific Concept. | | Prognosis | Forward chain: "What happens next?" | | Diagnosis | Backward chain: "What caused this?" | +----------------------------+------------------------------------------------+ =============================================================================== [5] DATA PERSISTENCE (I/O) The system utilizes localStorage for automatic session saving.
